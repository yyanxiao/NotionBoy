# NotionBoy

NotionBoy 是一个基于微信公众号和 Notion 的快速笔记应用，也是你的 AI 助手。


## Features

- 连接微信和 Notion，快速创建笔记
- 支持标签
- 支持剪辑网页全文
- 支持 ChatGPT 机器人，自动回复
- 支持 Z-Library 搜索，快速查找电子书
